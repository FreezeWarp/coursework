C Coursework

All coursework is available on GitHub; this is a highlight of material covered in the C class, without all the filler. Also check out my Blackjack code, which was the final project for this class.

  • Assignment 4 - Basic Input/Functions - Asks the user to input the dimensions of a rectangular prism. It then invokes user-defined functions to calculate its surface area, perimeter, volume, and chord.
    •
  • Exercise 17 - Is Factor Function - Basic input/output that determines if one inputted number is a factor of another. Uses modulus, because it'd be insane to calculate it any other way.
    •
  • Exercise 19 - printBinary Function - Implements a printBinary function using three different (but all inefficient) methods:
    • By using BAND
    • By subtracting from powers
    • With integer division (and supporting all bases)
    •
  • Exercise 21 - Basic Pointers - Implements a subset of Assignment 4 with pointers.
    •
  • Exercise 24 - Character Arrays - Converts a string to uppercase. See Exercise25 for same function using pointer notation.
    •
  • Exercise 28 - Tic-Tac-Toe - Implements code to support a tic-tac-toe game. Has logic to check if a winner exists, a function to print the board, and a function to make a move.
    •
  • Exercise 31 - Malloc and Sorting - Allocates storage dynamically and sorts inputted user values.
    •
  • Exercise 32 - Multidimensional Arrays and Malloc - Basic example of allocating a three-dimensional array with Malloc, and then printing it.
    •
  • Exercise 34 - Binary Writes - Writes float values to a binary file.
    •
  • Exercise 41 - Unions - Uses a union to determine if the system is little- or big- endian..
    •
  • Exercise 43 - Structs - Sorts structs based on a hash (more or less) of multiple properties and qsort.
    •
  • Exercise 47 - Linked Lists - Implements a single-linked struct.
